Posted on 2018-05-12 | In 中文 | | Visitors:
This is an solution about how to post my R code and result on Hexo blog

Blogdown

  • 现实情况：Hexo blog已经搭建很久，不想在新建一个网站
  • 需求：
    • 只是想在某些post里体现R代码；
    • 想让post显示在主页;
    • 想在博客搜索栏里可以搜到R project的内容;
    • 排版尽量美观；
  • 困难：blogdown似乎不适用我的情况。如果有人知道可以用blogdown解决的方案，可以邮件给我或者在下方留言。十分欢迎~
