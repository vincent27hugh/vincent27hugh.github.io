如何管理文献:Mendeley小技巧

Posted on 2017-11-12 | In 中文 | | Visitors:
This is some tips on how to manage your documents of reference papers.

PhD的生活基本就是由看文献，看网课和码代码组成。因此，关于如何管理文献，我最近花了很多时间探索。以下是本人血和泪的经验总结。

首先我不得不推荐一个强大的文献管理软件Mendeley（Wikipedia|Mendeley）。Mendeley教程可参见：

  • Mendeley|Videos and Tutorials
  • Mendeley 特色與操作使用

以下是本人的工作环境：

  • OS EI Capitan Version 10.11.6
  • Mendeley Desktop Version 1.17.11
  • Google Chrome Version 61.0.3163.100 (Official Build) (64-bit)

Mendeley小技巧

Personal Web Space

每个人注册了Mendeley账号之后都可以获得免费的2G云盘（Personal Web Space），需要额外的空间则需要升级账号。点击Preferences->General，你可以查看自己的网盘用量

File Organizer & Dropbox

你可以指定一个文件夹，然后将文件都copy到这个文件夹里，例如我都将自己的文献放到一个名为Reference的文件夹里。同时，你可以将此文件夹（例如我的Reference文件夹）放到Dropbox（当然，你也可以使用其他云盘）里，进行同步。点击Preferences->File Organizer->Organize my file，在Copy files to一栏进行设置。

同时如果你不仅仅希望在Mendeley Desktop上组织整理你导入的文献，还希望在物理上对文献进行组织，你可以勾上Sort files into subfolders一栏。例如，在Folder Path里，你选择Year和Journal，那么Mendeley会按照年份和期刊将导入的文件进行组织、分类。导入文件会被存放在路径.../Reference/Year/Journal/下。1

勾上Rename document files一栏，Mendeley会自动帮你对文件进行重命名。例如，在File name一栏里，你选择Title和Author，那么Mendeley会按照Title-Author.pdf的模式为你的文献重命名。

Watch Folder

点击Preferences->File Organizer->Organize my file，设置完自己的主文件夹（例如Reference）后，你可以到Preferences->Watched Folders里，将此文件夹（例如Reference）勾上，这样之后你直接添加文件到此文件夹里，Mendeley会自动同步。同时你也可以额外设置Watched Folders。

Tidy Up

当你需要整理你的Watched Folder时，到Preferences->File Organizer->Organize my file的Copy files to一栏里点击Tidy Up。Mendeley会讲未导入的文件Copy到Archive的文件夹里，这样就方便你将他们批量导入了。

关于导入Mendeley后，原文件可不可以删除的问题，经过本人试验，我认为是可以删除的。因为Mendeley会在你之前指示的路径下生成与原文件独立的相同文件。

bibtex

在Preferences->Bibtex里，你可以勾上Enable bibtex syncing，并且选择创建.bibtex文件的形式。例如，如果你选择的是Create one BibTex file for my whole library，并在Path里设置你存放.bibtex文件的路径，那么Mendeley会在该路径下创建记录着所有你导入的文献的.bibtex文件。

倘若你并不知道什么是.bibtex文件，那么可以参见我的另一篇文章用Sublime Text 3管理文献。

Check for Duplicate

如果你想检查导入文件是否存在重复，那么你可以点击Tools->Check for Duplicate进行查重，Mendeley会给出详细信息（例如Confidence）帮助你确认是否需要Merge相似的几个文件。

Files

观察Detail栏里的File里的文件，通过Open Containing Folder可以查看文件所在文件夹。检查：

  • 是否至于一个附件
  • 附件是否为PDF文件

如果一切正常，在View模式是Library as Table模式下的文件格式一栏里应该显示的是PDF符号。

插件Chrome

你可以再Chrome里现在Mendeley的插件Mendeley Importer，绑定你的账号后，你可以直接在文献页面点击Mendeley Importer，它会自动帮你导入该文献，并下载该文献的PDF文件。

Mac alias

除了使用Mendeley Desktop，我也十分喜欢Mac OS的Alias功能（类似Windows的快捷图标），通过快捷键Command+L，我可以为文件创建.alias文件。再将这个.alias文件粘贴复制到其他我想要存放的文件夹下，我就可以只用在电脑上储存一份文件，却可以在多个文件夹下访问它。优点是:

  1. 节省空间；
  2. 对文献的标记都可以集中到同一份文件上；
  3. 对原文件的路径进行改变，不会影响.alias文件的访问（这一点似乎比Windows的快捷图标要强大）。

其他

修改Reference文件夹里的文件名以及删除Reference文件夹里的文件似乎没用，同步以后又会恢复。所以要在Mendeley里做修改。

后记

Mendeley十分强大，但是它并不完美。
知乎上的一个回答2我觉得说得十分棒，最好的文献整理方法还是自己看完文献之后的Notes。我个人喜欢用Sublime Text 3 + Excel（当然你可以用更高级的Tool）整理自己的读后感。工具永远只是工具，只有花时间去看文献才是最直接的办法。
